Tomogachi Privacy Policy

Tomogachi is a Chrome extension that provides a small virtual pet game directly in the browser.

Tomogachi does not collect, sell, share, or transmit personal data to the developer, third parties, analytics services, advertising networks, or remote servers.

The extension stores game progress locally in the user’s browser using Chrome extension storage. This local data may include the pet’s name, stage, status, timers, preferences, and other game state needed for the virtual pet to function correctly.

This data remains on the user’s device and is used only to operate the Tomogachi game. The extension does not access browsing history, website content, personal communications, location, financial information, health information, authentication information, or payment information.

Tomogachi does not use remote code. All scripts, images, icons, styles, and audio behavior are included locally in the extension package.

Users can remove locally stored extension data by uninstalling the extension from Chrome.
